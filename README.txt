This is project Mendel for Sales Force Effectiveness in the Pharma Life Sciences Industry.
Based on python 3.7 with Flask Web Framework.

Development Setup Requirements.
1. Anaconda - Spyder, Anaconda Prompt.
2. Notepad++.

Python Libraries Required.
1. pandas
2. flask
3. flask-wtf
4. flask-login
5. flask-sqlalchemy
6. numpy
7. scikit-learn
8. matplotlib
9. seaborn
10. wtf-forms
11. statsmodels
 

Steps involved in running the tool
1. Firstly open the Anaconda Prompt from the start menu

2. Create a virtual environment for the code setup if not already created
    conda create --name mendel python=3.7

3. After creating the virtual environment choose the environment
    conda activate mendel
	
4. Install the required libraries
    conda install pandas

    Similarly install the other libraries
